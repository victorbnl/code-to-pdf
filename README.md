# Code to PDF

Writes source code into a PDF document.

## Requirements

- `pdflatex`

## Installation

```
pip install git+https://github.com/victorbnl/code-to-pdf
```

## Usage

```
code-to-pdf -s one-dark source.py document.pdf
```
